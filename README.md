# 🍅 Pomodoro Timer (JavaScript)

A simple and interactive Pomodoro Timer built with **Vanilla JavaScript**, using the **DOM API** and **LocalStorage** to enhance productivity and persist user settings.

https://gkglau.github.io/Pomodoro/

## 🚀 Features

- ⏱️ Start, pause, and reset timer
- 🔁 Automatic cycle between:
  - Work sessions
  - Short breaks
  - Long breaks
- 💾 Save user preferences using LocalStorage
- 🎯 Customizable durations
- 🔔 Notification sound when session ends
- 📱 Responsive and clean UI

## 🧠 How It Works

The Pomodoro Technique is a time management method that breaks work into intervals:

1. Work for 25 minutes
2. Take a short break (5 minutes)
3. Repeat 4 cycles
4. Take a long break (15 minutes)

This app automates the cycle and keeps track of your progress.

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- DOM Manipulation
- LocalStorage API
